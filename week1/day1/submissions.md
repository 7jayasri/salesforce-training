Important questions

 ## 1. What problem does Salesforce solve?
Salesforce helps companies manage customer information and business activities in one place.
Without Salesforce, customer data is often scattered across spreadsheets, emails, and different systems, which creates confusion and delays. 
Salesforce solves this by providing a centralized platform to track customers, sales, support, and communication efficiently.

 ## 2. What is CRM?
CRM stands for Customer Relationship Management.
It is a system used by businesses to store customer details, manage interactions, track sales, and improve relationships with customers.
CRM helps companies provide better service and maintain organized customer data.

## 3. What is an Object in Salesforce? Give one example.
An Object in Salesforce is used to store specific types of information, similar to a table in a database.
For example:
A Student Object can store student details like name, roll number, department, and phone number.
Salesforce also has standard objects like Account, Contact, and Lead.
Each entry inside an object is called a record.

## 4. Difference between Salesforce Admin and Developer
Salesforce Admin:
A Salesforce Admin mainly manages and configures Salesforce using built-in tools without much coding.
Admins handle users, security, reports, workflows, and data management.
Salesforce Developer:
A Salesforce Developer builds custom functionality using programming languages like Apex and technologies like Lightning Web Components (LWC).
Developers create custom applications, automation, integrations, and advanced business logic.
In simple terms:
Admin → configuration using clicks
Developer → customization using code

## 5. Suggest one real-world application that can be built using Salesforce.
One real-world application that can be built using Salesforce is a College Management System.
This application can manage:
     Student records
     Faculty details
    Attendance tracking
     Course management
      Fee payments
      Notifications and communication
Salesforce can help automate and organize all these activities in a single platform.
